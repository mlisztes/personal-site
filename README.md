# Personal Website

My personal website – built with React. 

See: [monikalisztes.com](https://monikalisztes.com/)

## Dependencies

Tested with node v16.

## Set up

To download the repository and install the dependencies, run the following commands:

```bash
git clone git://github.com/mlisztes/personal-site.git
cd personal-site
npm install
```

## Run

To run the app in development mode, run the following commands:

```bash
npm start
```

Your web browser should automatically open to `<ip>:<port>:<path>`, default: [http://localhost:3000/](http://localhost:3000/).

## Deploy to GitHub Pages

E.g., follow [this YouTube tutorial](https://www.youtube.com/watch?v=K5DTIf-jWhk) created by [@antonputra](https://github.com/antonputra).
